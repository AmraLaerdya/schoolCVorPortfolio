Game Memory Manipulation Tool (C# / DirectX 12) or in short CS2 Cheats

Developed in C# using Visual Studio, this project puts a strong emphasis on object-oriented programming (OOP) principles.
The tool scans the system's RAM to locate a specific game process and extract key in-memory data such as player location, health, orientation, and more. Once the game is identified, the program enters a monitoring loop that continuously checks whether a match has started.
Upon detecting an active match, the tool scans for all players on the server. Depending on user-selected options, it can perform various tasks such as:

  • Pinging enemy players on radar
  • Automated shooting and aiming
  • Drawing skeletal overlays using DirectX 12 and vector math
